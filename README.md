## Getting Started

1. First, to install this web app: 

```bash
yarn install
```

2. to run this web app:

```bash
yarn dev
```
and you need to wait until you find your url location like this

```
http://localhost:3000
```

#### included packages in this Web App

- heroicons
- tailwindcss


***to install heroicons***
```
yarn add @heroicons/react
```
